# Pose Coach — Official Website

Official one-page marketing website for [Pose Coach](https://apps.apple.com/tw/app/pose-coach/id1589037753), an iOS app for AI-powered sports motion analysis.

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- next-intl (zh / en)

## Development

```bash
npm install
npm run dev
```
